# KOROLI OS Build + Security Resources

This project is a minimal C + Assembly hobby OS starter. For deeper networking/security features, use these official resources.

## OS Development

- OSDev Bare Bones: https://wiki.osdev.org/Bare_Bones
- GRUB Manual: https://www.gnu.org/software/grub/manual/grub/grub.html
- QEMU i386 system docs: https://www.qemu.org/docs/master/system/target-i386.html

## URL/API Testing and Security Tools

- cURL manpage: https://curl.se/docs/manpage.html
- OWASP ZAP docs: https://www.zaproxy.org/docs/
- Nmap reference guide: https://nmap.org/book/man.html
- Nuclei overview: https://docs.projectdiscovery.io/opensource/nuclei/overview
- urlscan API docs: https://urlscan.io/docs/api/
- VirusTotal API docs: https://docs.virustotal.com/reference/overview
